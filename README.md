# FPS Game Prototype

This project is a simple first-person shooter prototype developed using **Unity (C#)** as part of a **club training program**.  
The focus of the project was to understand and implement core gameplay systems rather than visual polish.

## Features
- Player movement with jumping, shooting, and reloading mechanics
- Enemy interaction and elimination logic
- Real-time scoring system with a **dynamic multiplier** that increases on consecutive enemy kills

## Learning Outcomes
- Gained hands-on experience with Unity scripting and gameplay logic
- Implemented real-time score updates and event-based game interactions
- Improved understanding of player control systems and game state handling
